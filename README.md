# Discord Music bot for my study

## Setup

First of all, please make `config.json`

```
//config.json

{
  "prefix": "!",
  "token": "--your bot's token--"
}

```

```
npm install
```

## Start bot

```
node .
```

## Commands

- !beep
  - Return 'Boop.'
- !ping
  - Return 'Ping!'
- !play { youtube video's link }
  - listen to youtube video in Discord's voice sever
